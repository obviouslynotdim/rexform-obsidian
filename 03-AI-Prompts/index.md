# 🤖 AI Prompt Library

> These are **Obsidian Copilot** custom prompt templates.  
> Open this vault in Obsidian and use them via **Settings → Copilot → Custom Prompts** or the `/` command palette.

---

## Available Prompts

| Prompt | What it does |
|--------|-------------|
| **Clip Web Page** | Clips a web page into a structured Obsidian note |
| **Clip YouTube Transcript** | Generates a full note from a YouTube video with summary, key takeaways & mindmap |
| **Emojify** | Sprinkles relevant emojis throughout your text |
| **Explain Like I Am 5** | Simplifies complex concepts into plain language |
| **Fix Grammar & Spelling** | Corrects grammar and spelling errors |
| **Generate Glossary** | Extracts and defines key terms from a document |
| **Generate Table of Contents** | Creates a TOC from headings |
| **Make Longer** | Expands and elaborates on existing content |
| **Make Shorter** | Condenses content while keeping key points |
| **Remove URLs** | Strips all URLs from text |
| **Rewrite as Tweet Thread** | Converts content into a Twitter/X thread |
| **Rewrite as Tweet** | Converts content into a single tweet (280 chars) |
| **Simplify** | Rewrites text in simpler, clearer language |
| **Summarize** | Generates a concise summary |
| **Translate to Chinese** | Translates selected text to Chinese |

---

## How to Use

1. Open **Obsidian** and switch to this vault
2. Select any text in a note (optional — some prompts work on the whole note)
3. Open the **Copilot chat** panel (`Ctrl+Shift+P` → Copilot)
4. Type `/` and select the prompt from the list

> **Prompts folder:** `03-AI-Prompts/` — Copilot is configured to read from here automatically.
